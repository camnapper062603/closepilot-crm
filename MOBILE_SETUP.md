# Mobile and PWA Setup

The repo includes:

- PWA manifest
- app icons
- service worker
- Capacitor Android/iOS projects
- mobile viewport Playwright tests

Commands:

```bash
npm run mobile:sync
npm run mobile:open:android
npm run mobile:open:ios
```

Manual owner tasks still required:

- Apple developer account
- Google Play developer account
- app signing certificates
- store screenshots/listings
- final native QA
